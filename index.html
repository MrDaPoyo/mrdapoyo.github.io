My creations.

I have made a couple of projects along the years. Here's a selection of my coolest so far:

  • ★ Indieseas was an "indie web" search engine that indexed websites based on 88x31 button relations, preventing corporate slop from getting indexed (ish).
    At its peak, it had over 20k indexed buttons! I'm very proud of this project.

  • ★ PoyoWeb was a static website hosting service. Even though it is now a joke, due to how bad it was, I learned so many things from it.
    It will always have a special place in my heart <3

  • (WIP) Farga is a work-in-progress git forge. It's lightweight. :P

  • Dofi is a fantasy console I wrote in C using Raylib. It's very buggy but I did learn a lot! Pointers made sense from then onwards.

  • GET REKT!! lets you prank your friends. You send them a link, which returns a customized meme video with publicly available data that looks dangeous.
    Behind the scenes, it uses FFmpeg to overlay timed text over the meme matching its background song's BPM.

  • Pictoroom Is a Pictochat clone that works online. Chat with your friends using little drawings.
